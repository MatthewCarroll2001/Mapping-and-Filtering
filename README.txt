##  THESE TWO PYTHON SCRIPTS REPRESENT USES FOR THE map() AND filter() FUNCTIONS    ##

"filter.py" shows how the filter() function can be used for:
    * Filtering through a number list
    * Filtering through a list of items by type

"map.py" shows how the map() function can be used for:
    * Iterating through a list an calling a function on each items
    * Mapping keys to index values for the sake of creating map collections

There are many more uses for both of these functions,
but these scripts show the basic ideas of both
